## 一、最快上手

### 第一步：确认环境

```bash
cd D:\Gamecreate\hw
pip install -r requirements.txt
tshark --version          # 确认 tshark 可用
```

### 第二步：查看网卡

```bash
py main.py --list-interfaces
```

输出：

   1. \Device\NPF_{XXX} (以太网)

```bash

```

## 二、默认行为详解

运行 `py main.py -i "以太网"` 时（不加任何其他选项）：

### ✅ 默认开启

| 功能 | 说明 |
|------|------|
| 全量抓包 | 抓取网卡上所有流量，不做任何过滤 |
| 确认攻击检测 | 18 条规则，9 大类全部启用 |
| 可疑流量检测 | 16 条规则全部启用，终端黄色显示 |
| 定期统计摘要 | 每 5 分钟自动输出一次 |
| Ctrl+C 最终报告 | 停止时输出完整统计 |

### ❌ 默认关闭（需加选项启用）

| 功能 | 说明 | 如何启用 |
|------|------|---------|
| 可疑流量导出文件 | 只在终端显示，不写文件 | `--suspicious-export` |
| IOC 自动提取 | 不提取威胁指标 | `--extract-ioc` |
| IOC 导出文件 | 不写 IOC 文件 | `--ioc-export` |
| 日志文件 | 终端输出不保存 | `-o` |
| 流量过滤 | 抓所有包 | `-f` / `--src-ip` 等 |
| 限定检测类别 | 9 类全检 | `--detect webshell,c2` |
| 限定时长 | 持续运行 | `--duration 3600` |

## 三、参数错误提示

所有参数错误都会给出清晰提示，不会出现 Python 报错：

### 不加 `-i`

```bash
py main.py
```
```
错误: 请指定网卡
  使用 -i <网卡名> 指定，或 --list-interfaces 查看可用网卡
  示例: python main.py -i "以太网"
```

### 无效 IP

```bash
py main.py -i "以太网" --src-ip "abc"
```
```
错误: 源 IP (--src-ip) 格式无效: 'abc'
  正确格式: 10.0.0.1 或 192.168.0.0/16

共 1 个参数错误，请修正后重试。
```

### 无效检测类别

```bash
py main.py -i "以太网" --detect "wrong_category"
```
```
错误: 无效的检测类别: wrong_category
  可选类别: c2, cmd_injection, deserialization, dir_traversal, file_upload, log4shell, sql_injection, webshell, xss

共 1 个参数错误，请修正后重试。
```

### 无效端口

```bash
py main.py -i "以太网" --port 99999
```
```
错误: 端口号无效: 99999（有效范围: 1-65535）
```

### 无效正则表达式

```bash
py main.py -i "以太网" --content "[invalid"
```
```
错误: --content 正则表达式语法无效: unterminated character set at position 0
```

### 不存在的规则文件

```bash
py main.py -i "以太网" --rules "no_such_file.yaml"
```
```
错误: 确认攻击规则 (--rules) 文件不存在: no_such_file.yaml
```

### 负数时长

```bash
py main.py -i "以太网" --duration -10
```
```
错误: 时长必须大于 0（你输入了: -10）
```

### 无效网卡名

```bash
py main.py -i "不存在的网卡"
```
```
错误: 网卡不可用: '不存在的网卡'
  使用 --list-interfaces 查看可用网卡
```

## 四、自动路径功能

以下三个选项支持**不指定路径**，自动生成带时间戳的文件名：

| 选项 | 不加 | 加但不写路径 | 加且写路径 |
|------|------|-------------|-----------|
| `-o` | 不保存日志 | 自动生成 `capture_20260619_143000.log` | 保存到你指定的路径 |
| `--suspicious-export` | 不导出可疑流量 | 自动生成 `suspicious_traffic_20260619_143000.txt` | 导出到你指定的路径 |
| `--ioc-export` | 不导出 IOC | 自动生成 `ioc_20260619_143000.json` | 导出到你指定的路径 |

### 示例

```bash
# 自动生成 3 个文件
py main.py -i "以太网" -o --suspicious-export --ioc-export --extract-ioc

# 生成的文件：
#   capture_20260619_143000.log              ← 终端日志
#   capture_20260619_143000_report.json      ← 详细报告（随 -o 自动生成）
#   suspicious_traffic_20260619_143000.txt   ← 可疑流量
#   ioc_20260619_143000.json                 ← IOC 数据

# 指定路径
py main.py -i "以太网" -o my_log.txt --suspicious-export my_sus.txt --ioc-export my_ioc.json --extract-ioc
```

## 五、各选项使用示例

### 5.1 `--suspicious-export` — 导出可疑流量

```bash
# 自动命名
py main.py -i "以太网" --suspicious-export

# 指定文件名
py main.py -i "以太网" --suspicious-export my_suspicious.txt
```

导出文件每行一条，格式：
```
时间 | 严重度 | src:port -> dst:port | 协议 | 指标摘要 | Wireshark过滤器 | grep关键字
```

**用法**：复制 Wireshark 过滤器列，粘贴到 Wireshark 过滤栏，直接定位数据包。

---

### 5.2 `--extract-ioc` + `--ioc-export` — IOC 提取和导出

```bash
# 提取 IOC，终端显示
py main.py -i "以太网" --extract-ioc

# 提取 IOC + 导出 JSON 文件
py main.py -i "以太网" --extract-ioc --ioc-export

# 指定导出路径
py main.py -i "以太网" --extract-ioc --ioc-export ioc_report.json
```

---

### 5.3 `-f` — BPF 过滤（底层过滤，最高效）

```bash
# 只抓 HTTP + HTTPS + DNS
py main.py -i "以太网" -f "port 80 or port 443 or port 53"

# 只抓到某服务器的流量
py main.py -i "以太网" -f "host 192.168.1.100"

# 排除 SSH
py main.py -i "以太网" -f "not port 22"

# 横向移动常用端口
py main.py -i "以太网" -f "port 445 or port 3389 or port 5985"
```

---

### 5.4 `--src-ip` / `--dst-ip` — IP 过滤

```bash
py main.py -i "以太网" --src-ip 10.0.0.50
py main.py -i "以太网" --dst-ip 192.168.1.0/24
py main.py -i "以太网" --src-ip 10.0.0.50 --dst-ip 192.168.1.100
```

---

### 5.5 `--port` / `--protocol` — 端口和协议

```bash
py main.py -i "以太网" --port 8080
py main.py -i "以太网" --protocol dns
py main.py -i "以太网" --protocol http
```

---

### 5.6 `--detect` — 限定攻击检测类别

```bash
# 只检测 Webshell + C2
py main.py -i "以太网" --detect webshell,c2

# 只检测 SQL 注入
py main.py -i "以太网" --detect sql_injection

# 多类别组合
py main.py -i "以太网" --detect file_upload,deserialization,log4shell
```

---

### 5.7 `--no-detect` — 关闭确认攻击检测

```bash
# 只看可疑流量，不看确认攻击
py main.py -i "以太网" --no-detect --suspicious-export
```

---

### 5.8 `--no-suspicious` — 关闭可疑流量检测

```bash
# 只看确认攻击，不看可疑流量
py main.py -i "以太网" --no-suspicious
```

---

### 5.9 `-o` — 保存日志

```bash
# 自动命名
py main.py -i "以太网" -o

# 指定路径
py main.py -i "以太网" -o capture_log.txt
```

---

### 5.10 `--duration` — 限定运行时长

```bash
py main.py -i "以太网" --duration 3600    # 1 小时
py main.py -i "以太网" --duration 600     # 10 分钟
```

---

### 5.11 `--quiet` — 静默模式

```bash
py main.py -i "以太网" --quiet
```

只输出告警和可疑流量，不输出 INFO 日志。

---

### 5.12 `--summary-interval` — 修改定期报告间隔

```bash
py main.py -i "以太网" --summary-interval 60   # 每 60 秒
```

---

### 5.13 `--no-color` — 关闭彩色

```bash
py main.py -i "以太网" --no-color
```

## 六、实战场景

### 场景 1：护网全面监控（推荐）

```bash
py main.py -i "以太网" --suspicious-export --extract-ioc --ioc-export --duration 28800
```

| 功能 | 状态 |
|------|------|
| 确认攻击检测 | ✅ 全部 |
| 可疑流量检测 | ✅ 全部 |
| 可疑流量导出 | ✅ 自动生成文件 |
| IOC 提取 + 导出 | ✅ 自动生成 JSON |
| 时长 | 8 小时 |

### 场景 2：定向排查可疑 IP

```bash
py main.py -i "以太网" --src-ip 10.0.0.50 --suspicious-export
```

### 场景 3：只看 Webshell 和 C2

```bash
py main.py -i "以太网" --detect webshell,c2 --no-suspicious
```

### 场景 4：检测内网横向移动

```bash
py main.py -i "以太网" -f "port 445 or port 3389 or port 5985 or port 135" --suspicious-export
```

### 场景 5：监控 Web 服务器

```bash
py main.py -i "以太网" -f "port 80 or port 443 or port 8080" --suspicious-export --extract-ioc
```

### 场景 6：只看可疑，不看确认攻击

```bash
py main.py -i "以太网" --no-detect --suspicious-export --quiet
```

### 场景 7：用导出的 Wireshark 过滤器回溯

```bash
# 复制导出文件中的 Wireshark 过滤器，粘贴到 Wireshark
# 或用 tshark 命令行：
tshark -r capture.pcap -Y "ip.addr==10.0.0.51 && tcp.port==8080 && http.request.uri contains /actuator"
```

## 七、功能开关速查表

| 功能 | 默认 | 开启 | 关闭 |
|------|------|------|------|
| 全量抓包 | ✅ | `-i "以太网"` | — |
| 流量过滤 | ❌ | `-f` / `--src-ip` / `--port` | 不加 |
| 确认攻击检测 | ✅ 全部9类 | 默认 | `--no-detect` |
| 限定攻击类别 | 全部 | `--detect webshell,c2` | 不加=全部 |
| 可疑流量检测 | ✅ 全部16条 | 默认 | `--no-suspicious` |
| 可疑流量终端显示 | ✅ | 默认 | `--no-suspicious` |
| 可疑流量导出文件 | ❌ | `--suspicious-export [路径]` | 不加 |
| IOC 提取 | ❌ | `--extract-ioc` | 不加 |
| IOC 导出文件 | ❌ | `--ioc-export [路径]` | 不加 |
| 终端日志保存 | ❌ | `-o [路径]` | 不加 |
| 彩色输出 | ✅ | 默认 | `--no-color` |
| 定期统计摘要 | ✅ 300秒 | 默认 | `--summary-interval 0` |
| 静默模式 | ❌ | `--quiet` | 不加 |
| 限定时长 | 持续 | `--duration 3600` | 不加 |
